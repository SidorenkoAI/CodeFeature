 Pac-Man Lite на Arduino

Упрощённая версия классического Pac-Man, где игрок управляет героем с помощью аналогового джойстика. Никаких призраков и лабиринтов — просто собирайте все точки на поле и наслаждайтесь игрой!

 Особенности

Управление джойстиком:  перемещение во всех направлениях (вверх, вниз, влево, вправо).

Без стресса: нет призраков, нет лабиринта, нет жизней — просто собирайте предметы.

Аппаратное обеспечение

Плата: Arduino Uno 

Дисплей: LCD 16x2

Управление: Аналоговый джойстик 

Дополнительно: 8 проводов и макетная плата.

Cхема подключения 
Порт Arduino

LCD SDA	A4
LCD SCL	A5
LCD VCC	5V
LCD GND	GND
Джойстик VCC	5V
Джойстик GND	GND
Джойстик VERT 	A0
Джойстик HORZ 	A1

Управление:

Поворот джостика – перемещение Pac-Man’а в соответствующую сторону.



Примечание:  сброс можно реализовать через внешнюю кнопку или перезагрузку платы.

Принцип работы:

 1.При старте на экране случайным образом генерируются фрукты и монеты

 2.Pac-Man появляется в центре поля.

 3. Игрок отклоняет джойстик – персонаж перемещается на один шаг в выбранном направлении 

 4.Если под Pac-Man  находится предмет – он исчезает.

Примечение: перед загрузкой на реальное устройство настоятельно рекомендуем проверить на Wokwi !!!

Загрузка на реальное устройство

Подключите компоненты по схеме.

Откройте код в Arduino IDE.

Установите библиотеку LiquidCrystal_I2C (через менеджер библиотек).

Выберите плату и порт, нажмите «Загрузить».
